
# go-gorm

Sample project to explore gorm Golang library for SQL

## Run

### SQLite sample

```shell script
make run DB_TYPE=sqlite
```

### MySQL sample

```shell script
make run-mysql
make run DB_TYPE=mysql
```

### PostgreSQL sample

```shell script
make run-postgres
make run DB_TYPE=postgres
```
